# core.ai
A conversational AI platform that uses natural language processing to create simple, helpful chatbots that understand and respond to user requests in real-time.

### Language Model
This project is powered by **Gemma 3 270M**.

I chose 270 million parameters to ensure high speed and low memory overhead, preventing VPS overload. Despite its size, it handles conversational logic effectively while running in a fraction of RAM required by larger models.

### Animations
I chose GSAP over Framer Motion for this project. Since GSAP became completely free (including premium plugins) in 2025, it has become the most powerful tool for high performance, timeline-based animations without the overhead of paid licensing or the React-only constraints of other libraries.
